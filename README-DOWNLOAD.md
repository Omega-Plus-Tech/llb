# LodhiCoin APK download

To enable **Download** on the landing page:

1. Build or obtain your Android APK.
2. Rename it to **`LodhiCoin.apk`**.
3. Place **`LodhiCoin.apk`** in this folder (same level as `index.html`).

All “Download” buttons and links will then offer the file for download instead of opening the Play Store.
